# Bengali News Website - Sample Images

This directory contains placeholder images for the news website.

Since we cannot include actual copyrighted images, you should:

1. Replace these placeholder images with your own news images
2. Recommended image dimensions:
   - News cards: 800x600px
   - Featured news: 1200x800px
   - Thumbnails: 400x300px

3. You can use free image sources like:
   - Unsplash (https://unsplash.com)
   - Pexels (https://www.pexels.com)
   - Pixabay (https://pixabay.com)

4. Or use placeholder.com URLs in the sample-news.json file:
   - Example: https://via.placeholder.com/800x600?text=News+Image

For now, update the image URLs in src/data/sample-news.json to use placeholder.com URLs:
- https://via.placeholder.com/800x600/4F46E5/FFFFFF?text=News+Image+1
- https://via.placeholder.com/800x600/10B981/FFFFFF?text=News+Image+2
- etc.
